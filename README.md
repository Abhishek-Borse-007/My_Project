# EMIPro

A deploy-ready static EMI calculator website with:

- Home page
- EMI calculator tool page
- FAQ page
- Examples page
- SEO-ready metadata

## Deploy

This project is static HTML, CSS, and JavaScript, so you can deploy it on:

- Netlify
- Vercel
- GitHub Pages
- Cloudflare Pages

## Before deployment

Replace `https://example.com` in the HTML files, `robots.txt`, and `sitemap.xml` with your real domain.
